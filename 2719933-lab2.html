Latest Projects

1. Task Management App

I built a task management app using JavaScript, HTML, and CSS. The app allows users to create, edit, and delete tasks while categorizing them by priority and due date. It helped me practice DOM manipulation, event handling, and basic local storage for saving data, and it gave me a practical understanding of full-stack concepts.

2. To-Do List Web App

I built a to-do list web app using JavaScript, HTML, and CSS. The app allows users to add, delete, and mark tasks as completed. I integrated local storage so the tasks persist even when the page is reloaded. This project helped me practice JavaScript functions, array manipulation, and DOM updates, while improving my skills in organizing data and handling user interactions.

3. Weather App

I created a weather application that pulls real-time data using an API. The app displays the current weather, temperature, and forecast based on the user's location. This project helped me deepen my understanding of APIs, asynchronous JavaScript, and handling user inputs effectively.
